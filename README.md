## Выполненные действия

### Пройден vimtutor (уроки 1–7)

В ходе прохождения учебника были изучены и отработаны следующие команды:

#### Навигация
- `h`, `j`, `k`, `l` — перемещение по тексту
- `w`, `b` — перемещение по словам
- `gg`, `G` — переход в начало/конец файла

#### Режимы
- Нормальный режим (по умолчанию)
- Режим вставки (`i`, `a`, `o`, `O`)
- Командный режим (`:`)

#### Редактирование
- `x` — удалить символ под курсором
- `dd` — удалить строку
- `dw` — удалить слово
- `yy` — копировать строку
- `p` — вставить после курсора
- `u` — отменить действие
- `Ctrl + r` — повторить отменённое

#### Работа с файлами
- `:w` — сохранить файл
- `:q` — выйти
- `:wq` — сохранить и выйти
- `:q!` — выйти без сохранения

#### Поиск и замена
- `/word` — поиск слова вперёд
- `?word` — поиск слова назад
- `n` — повторить поиск
- `:s/old/new` — замена в строке
- `:%s/old/new/g` — глобальная замена
